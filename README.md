# OpenClaw Learn

📚 **Documentation & educational hub for OpenClaw** — the self-hosted multi-channel AI agent gateway.

**Live site:** [https://itziktdk.github.io/openclaw-learn/](https://itziktdk.github.io/openclaw-learn/)

## About

OpenClaw Learn is a Microsoft Learn-inspired documentation site for OpenClaw. It features:

- 🗂️ **Organized categories** — Getting Started, Installation, Configuration, Skills, Channels, Advanced, Community
- 🔍 **Client-side search** (Ctrl+K)
- 🌓 **Dark mode** with system preference detection
- 📱 **Fully responsive** — mobile hamburger menu
- 📋 **Code blocks** with copy button and syntax highlighting
- 🧭 **Breadcrumb navigation** and scrollspy table of contents
- ⬅️➡️ **Previous/Next article navigation**
- ✏️ **Edit on GitHub** and **Report Issue** links
- 📖 **Reading time estimates** and last updated dates
- 💡 **Info/Warning/Tip callout boxes**
- 📊 **Step-by-step numbered instructions**

## Content

Real, useful documentation covering:
- What is OpenClaw?
- Quick Start guide
- Ubuntu/Debian installation
- Docker installation  
- Azure VM deployment
- Telegram channel setup
- WhatsApp channel setup
- Skills system overview
- Creating custom skills
- And more...

## Tech Stack

- Single `index.html` — zero build step, zero dependencies
- Pure HTML/CSS/JS with embedded content
- Google Fonts (Inter + JetBrains Mono)
- LocalStorage for theme preference
- Hash-based routing
